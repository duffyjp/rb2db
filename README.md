# rb2db
Write CPU and Memory info to InfluxDB with Ruby

## Prerequisites
  Influx DB
  https://docs.influxdata.com/influxdb/

  Grafana (Optional)
  https://grafana.com/


## Install
    $ gem install rb2db
    
## Quick Start
    $ INFLUX_HOST='ip_or_hostname_here' rb2db
